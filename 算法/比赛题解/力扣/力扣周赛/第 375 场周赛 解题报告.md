## [T1. 统计已测试设备](https://leetcode.cn/contest/weekly-contest-375/problems/count-tested-devices-after-test-operations/)

思路: 差分思维

```java
class Solution {
    public int countTestedDevices(int[] batteryPercentages) {
        // 采用类似差分的思想
        int ans = 0;
        for (int v: batteryPercentages) {
            if (v - ans > 0) {
                ans++;
            }
        }
        return ans;
    }
}
```
